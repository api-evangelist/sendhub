# SendHubThis is a repo for managing machine readable artifacts for the [SendHub APIs](https://www.sendhub.com). The goal is to maintain and evolve an OpenAPI, Postman Collection, and APIs.json index for the [SendHub APIs](https://www.sendhub.com).They are a constant work in progress, and will be updated on a regular basis when new information is added, or more of the specifications are supported.Feel free to submit a pull request to evolve any of the artifacts, and leave GitHub issues if you have any questions, comments, or feedback.- [Website](https://www.sendhub.com)- [Developer](https://www.sendhub.com)- [Twitter](https://twitter.com/#!/SendHub)- [GitHub](https://github.com/Sendhub)The [APIs.json](https://github.com/api-evangelist/sendhub/blob/master/apis.json) index provides links to all the relevant links for the SendHub APIs, including the OpenAPI and Postman if they are present.
